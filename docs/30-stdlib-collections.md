# Standard Library: Collections

Common data structures.

## Vector

Dynamic array implementation.

```bpl
# Example usage (hypothetical API)
local v: Vector<int> = Vector.new<int>();
v.push(10);
```

## Map

Key-value store.

```bpl
local m: Map<string, int> = Map.new<string, int>();
m.set("key", 100);
```
