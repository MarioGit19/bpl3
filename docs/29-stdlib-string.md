# Standard Library: String Utilities

Functions for manipulating strings.

## Common Functions

- `strlen(s: string) ret int`
- `strcpy(dest: *char, src: string) ret *char`
- `strcat(dest: *char, src: string) ret *char`
- `strcmp(s1: string, s2: string) ret int`

_(Note: These are typically wrappers around C standard library functions)_
